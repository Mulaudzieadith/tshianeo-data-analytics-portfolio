# 📊 TikTok Claims Classification Analysis

### 📌 Project Overview
This project analyzes TikTok user data to prepare for a machine learning model that classifies videos as either claims or opinions.

### 🎯 Objectives
- Inspect and understand the dataset  
- Identify key variables  
- Assess data quality  
- Prepare data for future modeling  

### 🛠️ Tools Used
- Python (Pandas, NumPy)
- Jupyter Notebook

### 📂 Dataset
- ~19,000 TikTok videos  
- Includes engagement metrics (views, likes, shares, comments)  
- Contains classification label: claim vs opinion  

### 🔍 Key Insights
- Claim and opinion data are relatively balanced  
- Claim videos are more associated with banned or scrutinized authors  
- Engagement patterns differ between claim and opinion videos  
- High engagement does not always indicate reliable content  

### 🚀 Next Steps
- Perform full exploratory data analysis (EDA)  
- Handle missing values and outliers  
- Build a classification model
