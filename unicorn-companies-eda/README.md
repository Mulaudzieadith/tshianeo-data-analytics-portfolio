# Unicorn Companies EDA

## Project Overview
This project explores a dataset of unicorn companies (private companies valued at over $1 billion). The goal is to analyze how companies reach unicorn status and identify patterns across industries.

## Business Problem
An investing firm wants insights into unicorn companies to guide future investment decisions. This analysis focuses on:
- how long it takes companies to become unicorns
- which industries produce higher-value companies

## Dataset
The dataset contains 1,074 unicorn companies and 10 columns, including:
- Company  
- Valuation  
- Date Joined  
- Industry  
- City  
- Country/Region  
- Continent  
- Year Founded  
- Funding  
- Select Investors  

## Tools Used
- Python  
- pandas  
- matplotlib  
- Jupyter Notebook  

## Key Steps
- Explored the dataset using pandas  
- Checked structure using `head()`, `shape`, and `info()`  
- Converted `Date Joined` to datetime format  
- Created `Year Joined` and `Years to Unicorn`  
- Cleaned the `Valuation` column  
- Grouped data by industry  
- Created bar charts  

## Analysis Questions
- How long does it take companies in different industries to become unicorns?  
- Which industries have the highest valuations?  

## Key Findings
- Growth speed varies across industries  
- Some industries reach unicorn status faster  
- Some industries have much higher valuations  

## Conclusions
Industry plays a major role in how quickly companies grow and how valuable they become.

## Recommendations
- Focus on fast-growing industries  
- Prioritize industries with high valuation potential
